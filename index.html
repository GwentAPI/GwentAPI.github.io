Endpoints

GwentAPI currently support the following endpoints. More will follow.

URL Method Description
https://api.gwentapi.com/v0/ GET Return a mapping of resources name to href.
https://api.gwentapi.com/v0/cards/ GET Return all cards.
https://api.gwentapi.com/v0/cards/:id GET Return a specific card.
https://api.gwentapi.com/v0/factions/ GET Return all card factions.
https://api.gwentapi.com/v0/factions/:id GET Return a specific card faction.
https://api.gwentapi.com/v0/glyphs GET Return all glyphs.
https://api.gwentapi.com/v0/glyphs/:id GET Return a specific glyph.
https://api.gwentapi.com/v0/patches/ GET Return all game patches.
https://api.gwentapi.com/v0/patches/:id GET Return a specific game patch.
https://api.gwentapi.com/v0/patches/latest GET Return the latest game patch.
https://api.gwentapi.com/v0/rarities/ GET Return all card rarities.
https://api.gwentapi.com/v0/rarities/:id GET Return a specific card rarity.
https://api.gwentapi.com/v0/types/ GET Return all card types.
https://api.gwentapi.com/v0/types/:id GET Return a specific card type.
